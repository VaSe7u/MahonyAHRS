from 3-axis accelerations and 3-axis angular velocities. The calculated Euler angles will be written to the references passed to the function (in radians). The passed acceleration values can be relative to each other, but the passed angular velocities must be in radians per second.

Calculates Euler angles...

from 3-axis accelerations, 3-axis angular velocities and 3-axis magnetic fields. The calculated Euler angles will be written to the references passed to the function (in radians). The passed acceleration and magnetic field values can be relative to each other, but the passed angular velocities must be in radians per second.

Parameters
[out]&yawthis reference will be set to the calculated yaw angle (in radians).
[out]&pitchthis reference will be set to the calculated pitch angle (in radians).
[out]&rollthis reference will be set to the calculated roll angle (in radians).
[in]axx-axis acceleration.
[in]ayy-axis acceleration.
[in]azz-axis acceleration.
[in]gxx-axis angular velocity (in radians per second).
[in]gyy-axis angular velocity (in radians per second).
[in]gzz-axis angular velocity (in radians per second).
[in]axx-axis magnetic field.
[in]ayy-axis magnetic field.
[in]azz-axis magnetic field.
